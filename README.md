## Dependencies

### Authentication & Authorization
- [Laravel Fortify](https://laravel.com/docs/fortify) - Authentication scaffolding
- [Spatie Laravel Permission](https://spatie.be/docs/laravel-permission) - Role and permission management
- [Spatie Laravel One-Time Passwords](https://spatie.be/docs/laravel-one-time-passwords) - OTP generation and validation

### Auditing & Tracking
- [Laravel Auditing](https://laravel-auditing.com/) - Model auditing
- [Eloquent Blameable](https://github.com/richan-fongdasen/eloquent-blameable) - Track who created/updated records

### Media & File Management
- [Spatie Laravel Media Library](https://spatie.be/docs/laravel-medialibrary) - Associate files with models
- [Filament Spatie Media Library Plugin](https://filamentphp.com/plugins/filament-spatie-media-library) - Media library integration for Filament

### Admin Panel & UI
- [Filament](https://filamentphp.com) - Admin panel and form builder

### Notifications
- [Laravel Slack Notification Channel](https://laravel.com/docs/notifications#slack-notifications) - Send notifications to Slack
